Deleting one or more email messages
History | Edit

  • To delete a message, select it and then press the Delete key.
  • To delete more messages, select them by pressing and holding the Shift key. Then Release the Shift key and select Delete Messages.
  • To delete a saved message, select Menu -> Saved Messages. Select the message you would like to delete and then press the Delete key. If necessary, delete the message from your message Inbox as well.
